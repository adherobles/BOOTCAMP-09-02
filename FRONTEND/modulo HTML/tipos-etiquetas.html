Tag En Bloque (block)

    Ejemplos:
  • div
  • h1 al h6
  • parrafos (p)
  • contenedors y elementos semanticos

  • span
  • mark
  • Tag para enriquecer texto
  • italic(i)
  • bold(b)

Compartir espacio con otros elementos del tipo Inline inline-block viewport, Aqui si puedes configurar las dimensiones (ancho y alto)

Ejemplos; botones, imagenes, videos, iframes, formularios
